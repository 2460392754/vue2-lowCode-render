# vue2_low_code

## Project setup

```
pnpm install
```

### Compiles and hot-reloads for development

```
# http://localhost:6689/editor.html
# http://localhost:6689/h5.html

pnpm run serve
```

### Compiles and minifies for production

```
# http://xxxx/editor.html
# http://xxxx/h5.html

pnpm run build
```
